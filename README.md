	* ปรับหน้า ConvertByPayor ให้ทำงานได้อย่างถูกต้อง

	* ปรับมาใช้ BackgroundWorkerProcess เพื่อไม่ให้หน้าจอทำงานค้าง และ สามารถบอกสถานะการทำงานโปรแกรมได้ละเอียดขึ้น
	* เพิ่มโหมด Cancel โดยโปรแกรมจะวนทำงานจนครบข้อมูลคนปัจจุบันให้เสร็จก่อน ถึงยกเลิกข้อมูลที่เหลือ พร้อมสรุปจำนวนการทำงานว่าทำไปเท่าไร จากทั้งหมดกี่คน
	* ปรับหน้าแสดงข้อมูลรายชื่อพนักงาน ให้ไฮไลท์สีเหลือง กรณีมีการอัพเดทสถานะว่า "ทำการลงทะเบียนในโปรแกรม Mobile เรียบร้อยแล้ว"
